# Dell Mini 10 Netbook Recovery & OS Migration

**Device:** Dell Inspiron Mini 10 (2010-2011)  
**Original OS:** Windows 7 Home Premium (factory installed)

## Problem
Received a donated netbook from a family member with an unknown 
login password and no recovery options available. Device also had 
a failed battery requiring constant AC power.

## Process
- Downloaded Offline NT Password & Registry Editor (pogostick.net) 
  and flashed it to a bootable USB
- Accessed BIOS boot menu to change boot order to USB
- Used the tool to reset the Windows 7 login credentials
- Used the Dell recovery partition (2011 factory image) to factory 
  reset the drive
- Installed Linux Mint as the replacement OS

## Outcome
Successfully repurposed a 15-year-old locked netbook into a 
functional Linux machine, extending the hardware's usable life.

## Skills Demonstrated
Bootable media creation, BIOS navigation, Windows credential 
recovery, factory reset via OEM recovery partition, Linux OS 
installation, legacy hardware troubleshooting
